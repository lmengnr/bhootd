# C++ CLI: CPU stats
This repo is for learning command line instructions through c++. 

## Project Tasks
- Learn how to get cpu usage in Linux.
- Write a c++ program to get cpu usage/statistics and save the results in JSON.
- Parse the data.
- Send the data to MQTT broker.
- Repeat.
